Pasien
1. Pasien dapat registrasi akun dan login ke aplikasi. (sudah)
2. Pasien dapat menginput keluhan awal. (sudah)
3. Pasien dapat melihat jadwal praktek dokter (hari & jam buka). (udh)
4. Pasien dapat mengambil nomor antrian secara online tanpa harus datang langsung. (udh)
5. Pasien dapat melihat posisi antrian secara real-time (misalnya: “Nomor 8 sedang dilayani”). (udh)
6. Pasien dapat melihat estimasi waktu tunggu berdasarkan pasien yang sedang dilayani. (udh, cuma hard code)
7. Pasien dapat membatalkan antrian bila tidak jadi datang. (udh)
8. Pasien dapat melihat riwayat kunjungan ke klinik. (udh, tapi masih blm fix)
9. Pasien dapat menerima notifikasi ketika antriannya hampir tiba. (hanya memberi tanda kalau sudah di panggil)
10. Jika pasien datang terlambat, maka antrian akan dipindahkan ke paling akhir (udh, namun di cancel oleh dokter blm ada notif jika di batalkan) 

Dokter
1. Dokter dapat login ke aplikasi menggunakan akun yang disediakan.  (udh)
2. Dokter dapat melihat daftar antrian pasien hari ini secara berurutan.  (udh)
3. Dokter dapat memanggil pasien berikutnya dengan menekan tombol “Panggil”.  (udh)
4. Dokter dapat menandai pasien telah selesai diperiksa.  (udh)
5. Dokter dapat melihat total pasien yang dilayani per hari.  (udh)

Admin / Asisten Dokter 
1. Admin dapat melihat daftar pasien dan antrian secara keseluruhan. (udh)
2. Admin dapat menambahkan atau menghapus antrian manual. (udh)
3. Admin dapat melihat laporan pasien per hari/minggu/bulan. (udh, tapi masih blm fix)
4. Admin dapat mengatur jadwal praktek dokter. (udh)
5. Admin dapat memanggil pasien berikutnya dengan menekan tombol “Panggil”. (udh)
6. Admin dapat mangatur status praktik (buka/tutup antrian dan praktek).(udh)
